项目说明：规范驱动开发项目


### 使用方式
https://github.com/Fission-AI/OpenSpec/blob/main/docs/workflows.md

1、新增需求

```
/opsx:propose 增加一个新的黑暗模式功能出来
# 或者
给我生成一个黑暗模式的功能出来，用openspec跑一遍，然后等我确认。
```

2、应用需求
```
/opsx:apply
```

3、归档需求
```
/opsx: archive
```
